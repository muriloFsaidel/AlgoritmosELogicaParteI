## Algoritmos e Lógica Parte I

  <p>O projeto foi concebido a partir da conclusão do curso Algoritmos e Lógica Parte I na udemy e ministrado pelo professor Cleison.<br/>
     Conteúdo do curso Algoritmos e Lógica Parte I que tem por objetivo colocar em prática conceitos e estruturas importantes na programação de modo geral, usando como instrumentos: Pseudocódigo e Javascript. A navegação dos mesmos está no github pages e repositório. 
   </p>
