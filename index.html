Exercícios práticos

Princípios Utilizados

Algoritmos

Todos os exercícios partem da lógica e dos Algoritmos criados, que em resumo contemplar os passos lógicos que usamos para resolver um problema ou executar uma tarefa.

Inferências

Todos os fluxos de algoritmo, são definidos pelas inferências que adotamos, por exemplo, o aluno passou de ano? informado as premissas que ele teve nota x e y e uma resultaram em uma média z > p, conclui-se que o aluno passou de ano.

Com base nisto, os algoritmos são construídos linha a linha, pensando em uma estrutura de sequência de ocorrências ou ações que tomada uma decisão apresenta um determinado resultado ou outro, podendo se repetir ou não.

Implementação

Os exercícios foram desenvolvidos usando pseudocódigo e transportados para o Javascript, para sair de um conceito para uma linguagem de programação, simulando um ambiente.

Javascript?

O javascript é a linguagem web que permite executar os programas do lado navegador, trazendo conteúdos dinâmicos (escolha de exercícios diferente), sem precisar instalar compiladores ou interpretadores específicos, por isso foi selecionado.
